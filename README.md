# Место

## Описание проекта

«Место» — это интерактивная веб-страница, созданная для визуального обмена интересными локациями и путешествиями. Проект представляет собой полноценный сервис, в котором пользователи могут делиться фотографиями живописных уголков, выражать симпатии с помощью лайков и управлять своей персональной галереей. Основной фокус работы — переход от статической верстки к интерактивному динамическому интерфейсу. С помощью JavaScript реализовано живое взаимодействие с пользователем: открытие модальных окон, обработка отправки форм, а также мгновенное обновление контента страницы без её перезагрузки.

**Функциональность и особенности:**

- Семантическая верстка на HTML.
- Адаптивная сетка карточек, подстраивающаяся под различные разрешения экранов (от 320px до 1280px).
- Всплывающие окна (попапы): реализованы формы для редактирования данных профиля пользователя и добавления новых карточек в галерею, а также отдельный попап для просмотра картинок в увеличенном масштабе.
- Интерактивные элементы галереи: возможность ставить лайки фотографиям и удалять добавленные карточки.
- Плавное открытие и закрытие всех всплывающих окон (попапов) за счет CSS-транзиций.
- Динамическое обновление текущего года в подвале сайта.

**Технологии:**

1. **HTML** — семантическая и логичная разметка структуры страницы.
2. **CSS** — стилизация, использование Flexbox и Grid-сеток, создание микроинтеракций (плавное изменение непрозрачности) и медиазапросы для адаптации под разные экраны.
3. **БЭМ** — строгая методология именования классов, которая обеспечивает независимость блоков и чистоту кода без использования сторонних фреймворков.
4. **JavaScript** — добавление интерактивности, работа с DOM-деревом и обработка событий пользователя. Реализована динамическая генерация карточек из массива данных с использованием тега `<template>`, а также обработка отправки форм с отменой стандартной перезагрузки страницы.
