# Место

## Описание проекта

«Место» — это интерактивная веб-страница, представляющая собой интерфейс с профилем пользователя и галереей фотографий. Главная цель этого проекта — переход от статической верстки к динамической. С помощью JavaScript реализована обработка пользовательских действий, работа с формами и изменение содержимого страницы без её перезагрузки.

**Функциональность и особенности:**

- Семантическая верстка на HTML.
- Адаптивная сетка карточек, подстраивающаяся под различные разрешения экранов (от 320px до 1280px).
- Интерактивные элементы: возможность ставить лайки фотографиям.
- Всплывающее окно (попап) для редактирования данных профиля пользователя.
- Динамическое обновление текущего года в подвале сайта.

**Технологии:**

1. **HTML** — семантическая и логичная разметка структуры страницы.
2. **CSS** — стилизация, использование Flexbox и Grid-сеток, создание микроинтеракций (плавное изменение непрозрачности) и медиазапросы для адаптации под разные экраны.
3. **БЭМ** — строгая методология именования классов, которая обеспечивает независимость блоков и чистоту кода без использования сторонних фреймворков.
4. **JavaScript** — добавление интерактивности, работа с DOM-деревом и обработка событий пользователя.
